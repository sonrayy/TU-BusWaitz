# TU-BusWaitz-Chatbot
TU BusWaitz is a Line chatbot which show the schedule of TU bus in different routes around the university.
# Echo Bot

An example LINE bot just to echo messages

## How to use

### Install deps

``` shell
$ npm install
```

### Configuration

``` shell
$ export CHANNEL_SECRET=YOUR_CHANNEL_SECRET
$ export CHANNEL_ACCESS_TOKEN=YOUR_CHANNEL_ACCESS_TOKEN
$ export PORT=1234
```

### Run

``` shell
$ node .
```

## Webhook URL

```
https://your.base.url/callback
```
